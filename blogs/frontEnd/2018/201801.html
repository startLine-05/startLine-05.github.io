前端学习基本知识简介

vuePress-theme-reco startLine    2018 - 2021

前端学习基本知识简介

startLine 2018-05-02 知识清单

# HTML

  • 首先要注意html标签的语义化，要理解“所有的html标签都是容器”这个概念，要求对浏览器文档对象的常用操作以及属性都要掌握
  • 然后是html5新特性，首先就是增强型表单，例如进度条、刻度尺等，各种表单元素的新属性等，还有音频和视频标签也要掌握， canvas、svg标签也要要掌握，现在很多网页端、移动端的都需数据展示、页面动画，很多都是使用canvas和svg来绘制的 然后是，Geolocation:地理定位，很多项目上的定位需求 都是使用js来获得当前所在的地理坐标，应用非常广泛了WebStorage浏览器存储、WebSocket长连接，一定要掌握

# CSS

  • Css部分，首先css的基本语法、盒模型、定位，浮动、文档流、优先级、权重这些一定要熟练掌握。
  • 熟悉viewport的概念 知道所谓的PC端的px和移动端的px的区别, 熟悉重排与重绘。
  • 熟悉常用的布局方式 如Flex 布局
  • Css3的transition和animation动画要熟练应用，因为在移动端非常多的css3的动画需求。推荐网络教程阮一峰的网络日志 » 动画教程

# JavaScript

  • 了解javaScript 语言的特性，如javaScript是一门脚本性语言，解释性语言，动态性，弱类型语言，面向对象，松散性语言。
  • javaScript 的基本类型，基本对象与方法。
  • this、new、词法分析、new 生成一个对象的过程、call & apply、继承、原型链。
  • 模块化:公共&私有方法（闭包)、lIFE、作用域链、递归、链式调用等，这些基本的知识点一定要掌握
  • ES6 的重点主要在于异步的处理，包括 Promise、Generator，yield、async、await、next等相关的方法，都要熟练

# JavaScript设计模式（只要求了解设计思维即可）

说到这里就要提到设计模式了，对于前端开发来讲最基本的设计模式，原型、观察者、单例、工厂模式这几个是最基本的要掌握的,虽然在实运用不多但是设计思维是进阶高阶的必经之路。 例如，jq就是单例模式，vue就是观察者模式这些常识性的东西一定要知道

# 计算机基础之 数据结构与算法 （基本功）

  • 数据结构是计算机存储、组织数据的方式。数据结构是指相互之间存在一种或多种特定关系的数据元素的集合。通常情况下，精心选择的数据结构可以带来更高的运行或者存储效率。数据结构往往同高效的检索算法和索引技术有关
  • 简单说，数据结构其实是数据之间的逻辑关系，栈、表、队列、树等等，它们反映的都是数据之间的基于逻辑关系而保存的结构，这个一定要掌握。 为什么优秀的前端可以和后端程序员一起制定接口规范呢?因为合适的数据结构，会提升更多的开发效率。
  • 有了数据，要用算法进行相应的处理，那么就要把数据组成为相应的结构。所以有了的数据结构，才会有相应的算法。也就是说，离开了算法，数据也没什么意义了，所以算法和数据结构一定要掌握。
  • 推荐的算法和数据题库 力扣

# 计算机网络

    1. http网络协议方面的，https、Tcp/ip等
    1. http三次握手、七层网络等，这些都需要掌握
    1. get、post请求的异同，请求头，请求体的知识
    1. 看懂网络瀑布图，常见的code 对应的含义

# 浏览器与优化

    1. 浏览器是如何生成dom树，渲染页面的顺序，请求的阻塞
    1. 熟悉 DOM,BOM 对象。
    1. js的性能优化、垃圾回收等等方面的知识，都要掌握
    1. 前端开发对于浏览器并发请求的处理等等
